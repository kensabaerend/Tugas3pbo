CILOK

Animasi yang saya buat ini terinspirasi dari makanan tradisional dari daerah Jawa Barat yaitu 'Cilok'. Dalam pembuatan Animasi ini menggunakan Software Macromedia Flash 8, Cilok Pernah mengikuti kompetisi Tingkat Jawa Barat yaitu Jabar Tanggap Corona (CovEducation).

BUAT AKUN

Web yang saya buat ini menyediakan tampilan serta tempat untuk memasukan data tetapi web belum dapat mengakses kedalam data base. Dalam pembuatan web ini menggunakan Software Notepad++, Untuk Sekarang pengerjaan web ini terhambat karena data terhapus.

ZEEV

Model 3 Dimensi yang saya buat ini bertujuan untuk mempresentasikan tekologi dengan bentuk manusia. Dalam pembuatan Model 3D ini menggunakan Vroid dan Photoshop, Untuk Sekarang pengerjaan Model 3D ini terhambat karena data terhapus.

WAYANG

Desain Poster yang saya buat ini terinspirasi dari salah satu kesenian tradisional Indonesia yaitu 'Wayang'. Dalam pembuatan Desain ini menggunakan Software Photoshop, Wayang Pernah mengikuti kompetisi Tingkat Kota Bandung yaitu Festival dan Lomba Seni Siswa Nasional (FLS2N).

Dalam pembuatan game ini menggunakan Software Game Maker, Sondah Edukasi Anak Milenial (SENDAL) Meraih Mendali Perak dalam Kompetisi Tingkat Nasional yaitu Festival Inovasi dan Kewirausahaan Siswa Indonseia (FIKSI).
